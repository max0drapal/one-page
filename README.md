# Název projektu
pivovar Kocour
# Žák
Max Drápal
# Stručný popis tématu (3–5 vět)
Webové představení pivovaru Kocour. Stránka bude na způsob one-page webu s sekcemi co ukážou co pivovar nabýzí a akce co se zde konají. Stánka bude interaktivní za pomoci JavaScriptu, ale zároveň přehledná. Cílem je vytvořit web, co ukáže lidem proč by měli pivovar navětívit.
# Cílová skupina
Obyvatelé Varnsdorfu a milovníci piva.
# Návrh obsahových sekcí
- o nás
- výběr piv a limonád
- ubytováni
- akce
- kontakty
# Návrh designu / wireframe
- barvy budou v souladu s logem a zároveň bude zachován prehledný a moderní design
- stránka bude mít přístupnost z PC i mobilu
- stránka bude beź reklam
- možnost registrace
# Použité technologie
- **front-end:** HTML, CSS, JavaScript
- **data**: JSON pro strukturu dat
- **verzování**: Github
